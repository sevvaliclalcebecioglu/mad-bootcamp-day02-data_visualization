Welcome to my **Data Visualization & Exploratory Data Analysis (EDA) Project**!  
This project demonstrates how to explore, clean, and visualize data using **Python** with libraries like **Pandas, Numpy, Matplotlib, Seaborn, Plotly**, and **NetworkX**.

---

## ðŸ”¹ What This Project Contains

1. **Data Exploration (EDA)**  
   - Viewing data with `head()`, `tail()`, `info()`, `describe()`.
   - Checking missing values and filling them with `fillna()`.
   - Understanding column distributions, correlations, and unique values.

2. **Data Cleaning & Imputation**  
   - Handling missing values using mean, median, mode, or placeholders.
   - Dropping columns with mostly missing data.
   - Replacing categorical values for clarity.

3. **Feature Engineering**  
   - Creating new features from existing columns (e.g., extracting day, month, year from a date).
   - Converting date columns to datetime and generating weekday/month names.
   
4. **Data Visualization Techniques**  
   - **Matplotlib:** Line, scatter, histogram, area, donut, romantic heart plot.  
   - **Seaborn:** Countplots, boxplots, pairplots, violin plots, swarmplots.  
   - **Plotly:** Interactive sunburst charts, scatter plots, sankey diagrams.  
   - **3D Plots:** Surface plots and scatter plots in 3D space.  
   - **NetworkX:** Social network visualization.

5. **Financial Data Analysis**  
   - Fetching live stock data with `yfinance`.
   - Visualizing closing prices and market trends.

6. **Google Trends Analysis**  
   - Compare search trends for topics over time using `pytrends`.
